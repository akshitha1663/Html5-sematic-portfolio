# HTML5 Portfolio

Simple personal portfolio built with plain HTML, CSS and JavaScript.

## Overview

- Static site with pages: `index.html`, `about.html`, `projects.html`, `contact.html`.
- Assets live in `assets/images/`, styles in `css/styles.css`, scripts in `js/main.js`.

## Project Structure

- index.html — Home page
- about.html — About page
- projects.html — Projects listing
- contact.html — Contact form or details
- assets/images/ — Images and icons
- css/styles.css — Main stylesheet
- js/main.js — Main JavaScript

## How to view

Open `index.html` directly in a browser, or run a simple HTTP server for local development:

Python 3:

```bash
python -m http.server 8000
# then open http://localhost:8000 in your browser
```

Node (http-server):

```bash
npm install -g http-server
http-server . -p 8000
```

## Development

- Edit files in the repository and refresh the browser.
- Keep images in `assets/images/` and update paths in HTML.
